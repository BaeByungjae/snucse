Deep Learning from Scratch
========
> 남들 다 할줄아는 딥러닝을 나도 하기위해, 만학의 꿈에 도전하는 김젼

```bash
# Install dependencies
pipenv install

# Execute jupyter notebook
pipenv run jupyter -- notebook --no-browser \
  --ip=$(dig +short myip.opendns.com @resolver1.opendns.com)
```
